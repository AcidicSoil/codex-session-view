# Component Reference Inventory

Centralized list of external component inspirations referenced across `PENDING/` notes. Use these when replacing or upgrading existing UI; if a component has already been adopted elsewhere, document it in future updates.

## Session Explorer Filters & Controls
- [Multi Selector – ui-layouts](https://www.ui-layouts.com/components/multi-selector): primary pattern to consolidate toolbar filters into a dense multi-select control.
- [Tags Input – ui-layouts](https://www.ui-layouts.com/components/tags-input): chip-based entry surface for additional filters or metadata tags.
- [Datetime Picker – ui-layouts](https://www.ui-layouts.com/components/datetime-picker): reference for explicit time-range selection in filters.
- [Toggle Layout Buttons – Systaliko](https://systaliko-ui.vercel.app/docs/ecommerce/toggle-layout): style inspiration for per-view toggles or filter presets.
- [Scroll Animation Trigger – Systaliko](https://systaliko-ui.vercel.app/docs/blocks/scroll-animations-rotate): potential hero/intro animation when the Session Explorer first appears.
- [Header Block – Systaliko](https://systaliko-ui.vercel.app/docs/blocks/header): navigation/header styling reference to keep filters aligned with top-level nav.

## Rule Inspector & Hook Gate
- [Accordion – ui-layouts](https://www.ui-layouts.com/components/accordion): structural inspiration for Hook Gate / Rule Inspector panels.

## Timeline Interactions & Effects
- [3D Pin – Aceternity](https://ui.aceternity.com/components/3d-pin): use for pointing from events to files/props.
- [Expandable Card – Aceternity](https://ui.aceternity.com/components/expandable-card): long event payload expansion pattern.
- [Animate UI Code Blocks](https://animate-ui.com/docs/components/animate/code): follow-up animation once timeline pop-outs expand fully.
- [Scroll Beam Animation – Animate UI](https://animate-ui.com/docs/components/animate/code): generic reference for timeline progress animations.
